перед запуском необходимо установить
pip install websockets

как запустить сервер: в папке server прописать команду 
uvicorn main:app --reload

после чего открыть три вкладки (host, player0, player1) в браузере из папки client/client.html

для корректной работы вкладку хоста(первую открытую) нельзя закрывать
